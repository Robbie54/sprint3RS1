CylinderDetector::CylinderDetector ( )

Constructor for the CylinderDetector node.

Initializes the CylinderDetector node by subscribing to the laser scan topic and setting up a publisher for visualization markers. The default cylinder diameter is set to 0.3 meters.

Fits a circle to a segment of laser scan data.

This function attempts to fit a circle to a set of points derived from laser scan data between the given start and end indices. It uses a least-squares method to compute the circle's center and radius. The points are transformed into the map frame before fitting.

Parameters
msgShared pointer to the received LaserScan message.
startThe index of the first point in the laser scan data to use.
endThe index of the last point in the laser scan data to use.
Returns
Circle The fitted circle containing the center (a, b), radius (r), and the number of iterations (j) performed during the fitting.

Checks if a given cluster of laser scan points forms a circular shape.

This function estimates the diameter of a cluster of points by using the law of cosines and compares it to the expected cylinder diameter. It also checks if the midpoint of the cluster is closer than the edge points, indicating a cylindrical shape.

Parameters
clusterA vector of range values representing the cluster.
angle_incrementThe angular increment between successive laser scan points.
Returns
true If the cluster forms a circular shape.
false If the cluster does not form a circular shape.

Publishes a visualization marker to represent the detected cylinder.

This function creates and configures a cylinder marker for RViz visualization and publishes it to the /cylinder_marker topic. The marker's position is set based on the detected point and its scale is adjusted to match the cylinder's diameter.

Parameters
pointThe 2D coordinates of the cylinder in the map frame.
diameterThe diameter of the detected cylinder.
